# Política de Privacidade — MAINTRIX Enterprise
**Versão:** 1.0 | **Data:** 11 de Julho de 2026
**Em conformidade com a Lei Geral de Proteção de Dados (Lei 13.709/2018)**

---

## 1. Dados Coletados

A MAINTRIX coleta os seguintes dados pessoais:

**Dados de Cadastro:**
- Nome completo
- Endereço de e-mail corporativo
- Telefone (opcional)
- Cargo/função na organização
- Disciplinas de trabalho

**Dados de Uso:**
- Registros de acesso (data, hora, endereço IP)
- Ações realizadas na Plataforma (logs de auditoria)
- Dados operacionais inseridos (ordens de serviço, inspeções, registros de manutenção)

**Dados Técnicos:**
- Tipo de dispositivo e navegador
- Sistema operacional
- Dados de geolocalização (apenas quando autorizado para funcionalidades de campo)

## 2. Finalidade do Tratamento

Os dados são tratados para as seguintes finalidades:
- Autenticação e controle de acesso ao sistema
- Execução das funcionalidades contratadas (gestão de manutenção)
- Geração de relatórios e indicadores de desempenho
- Auditoria de segurança e conformidade
- Suporte técnico ao usuário
- Melhorias na Plataforma

## 3. Base Legal

O tratamento dos dados pessoais está fundamentado nas seguintes bases legais da LGPD:
- **Art. 7, V** — Execução de contrato: dados necessários para prestação do serviço contratado
- **Art. 7, IX** — Interesse legítimo: logs de segurança e auditoria
- **Art. 7, I** — Consentimento: para dados opcionais e comunicações

## 4. Compartilhamento

Os dados NÃO são comercializados ou compartilhados com terceiros para fins de marketing. Podem ser compartilhados com:
- Provedores de infraestrutura em nuvem (armazenamento e processamento)
- Autoridades competentes, quando exigido por lei ou ordem judicial

## 5. Armazenamento

Os dados são armazenados em servidores seguros com as seguintes proteções:
- Criptografia em trânsito (TLS/HTTPS)
- Controle de acesso baseado em papéis (RBAC)
- Isolamento multi-tenant (dados de cada empresa são segregados)
- Backups regulares
- Logs de auditoria para todas as ações

## 6. Segurança

A MAINTRIX implementa medidas técnicas e organizacionais para proteger os dados:
- Autenticação por credenciais com hash seguro (bcrypt)
- Rate limiting para prevenção de ataques de força bruta
- Security Headers (HSTS, X-Frame-Options, CSP)
- Monitoramento contínuo de acessos
- Política de senhas com complexidade mínima

## 7. Direitos do Titular

Em conformidade com a LGPD, o titular dos dados tem direito a:
- **Acesso:** Solicitar cópia dos seus dados pessoais
- **Correção:** Solicitar correção de dados incompletos ou inexatos
- **Eliminação:** Solicitar a exclusão dos seus dados pessoais
- **Portabilidade:** Solicitar a transferência dos seus dados
- **Revogação:** Revogar o consentimento a qualquer momento
- **Informação:** Ser informado sobre com quem seus dados são compartilhados

Para exercer estes direitos, entre em contato: **privacidade@maintrix.com.br**

## 8. Retenção e Exclusão

- **Dados ativos:** Mantidos enquanto o contrato estiver vigente
- **Após encerramento:** Dados mantidos por 90 dias para exportação, depois excluídos
- **Logs de auditoria:** Retidos por 5 anos conforme obrigações legais
- **Dados de consentimento:** Retidos indefinidamente para comprovação legal

## 9. Cookies e Tecnologias

A Plataforma utiliza:
- **SessionStorage:** Armazenamento do token de sessão (não persiste após fechar o navegador)
- **IndexedDB:** Cache de dados para operação offline (PWA)
- **Service Workers:** Funcionamento offline da aplicação

Não utilizamos cookies de rastreamento ou publicidade.

## 10. Contato do Encarregado (DPO)

Para questões relacionadas à proteção de dados:
- **E-mail:** privacidade@maintrix.com.br
- **Responsável:** Encarregado de Proteção de Dados — MAINTRIX Tecnologia Ltda.

## 11. Alterações

Esta Política pode ser atualizada periodicamente. Quando houver alterações significativas, os usuários serão notificados e um novo aceite será solicitado. O histórico de versões está disponível no sistema.
